### ðŸ“Œ What I Learned
- **What Web Hosting Is**
  - Web hosting is a service that stores website files on a server and makes them accessible on the internet.
  - Without hosting, a website cannot be viewed publicly.

- **Why Hosting Is Important**
  - Hosting allows users worldwide to access a website using a domain name.
  - It provides storage, uptime, security, and performance for websites.

- **Types of Web Hosting**
  - **Shared Hosting**
    - Multiple websites share the same server.
    - Low cost and suitable for beginners.
  - **VPS (Virtual Private Server)**
    - Dedicated resources within a shared server.
    - Better performance and control.
  - **Dedicated Hosting**
    - A full server for one website.
    - High performance and high cost.
  - **Cloud Hosting**
    - Uses multiple servers for better scalability and reliability.

- **How to Choose the Right Hosting**
  - Depends on website size, traffic, budget, and technical skills.
  - Beginners should start with shared hosting and upgrade later.
